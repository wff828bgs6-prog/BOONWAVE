# BOONWAVE PWA — автоматическое обновление

Патч предназначен для текущей PWA-сборки BOONWAVE.

## 1. sw.js

Если в `sw.js` уже есть обработчики `install` и `activate`, не создавайте вторые без необходимости.

В существующий `install` добавьте:

```js
event.waitUntil(self.skipWaiting());
```

В существующий `activate` добавьте:

```js
event.waitUntil(self.clients.claim());
```

Файл `sw-full-example.js` показывает безопасную общую структуру с `Promise.all`.

## 2. Регистрация service worker

Замените текущий блок регистрации содержимым `service-worker-registration.js`
или перенесите из него:

- слушатель `controllerchange`;
- флаг `refreshing`;
- `window.location.reload()`.

Слушатель должен быть установлен до вызова `navigator.serviceWorker.register()`.

## Что произойдёт

После публикации изменённого `sw.js` браузер установит новую версию,
она сразу пропустит фазу ожидания, возьмёт управление открытыми страницами,
после чего приложение один раз перезагрузится и откроет новую сборку.

## Важно

При каждом деплое меняйте имя/версию кэша, например:

```js
const CACHE_NAME = 'boonwave-v8';
```

Иначе старые файлы могут оставаться в существующем кэше.

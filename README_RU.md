# BOONWAVE iPhone MVP v6

Обновлённая iPhone/PWA-ветка BOONWAVE на базе Version 5.

## Главное изменение

Для корневого элемента типа **«Проект»** реализована отдельная расширенная карточка. Другие типы элементов продолжают использовать универсальную структуру Version 5.

## Структура проекта

- название проекта;
- приоритет: высокий / средний / низкий;
- статус: стадия подготовки / в работе / на паузе / отменён;
- адрес объекта;
- количество позиций;
- дата подписания;
- бюджет;
- ожидаемая прибыль;
- сумма и дата аванса;
- сумма и дата остатка;
- имя основного контактного лица;
- номер телефона с кнопкой звонка;
- электронная почта с кнопкой письма;
- предварительная информация по проекту;
- фото или логотип карточки проекта;
- люди в проекте и их роли;
- изображения, PDF-документы и другие файлы.

## Материалы проекта

Вложения разделены на три горизонтально прокручиваемых раздела:

- изображения;
- PDF-документы;
- другие файлы.

Нажатие открывает изображение или PDF. Удержание около 0,5 секунды открывает меню: открыть, отправить, скачать или удалить.

## Свёрнутый проект

При сворачивании проекта отображается компактная миниатюра с фотографией или логотипом. Цвет обводки зависит от статуса:

- белый — стадия подготовки;
- зелёный — в работе;
- жёлтый — на паузе;
- красный — отменён.

## Создание корневого элемента

Кнопка **«Корневой элемент»** открывает выбор типа:

- проект;
- цель;
- человек;
- этап;
- идея.

Выбор «Проект» сразу открывает специализированную карточку проекта.

## Контакты

Форма контакта появляется после нажатия «+ Новый» или при редактировании существующего человека. В списке есть отдельные действия просмотра и изменения.

## Установка и тестирование на iPhone

1. Загрузите содержимое архива на Netlify.
2. В корне публикации должны находиться `index.html`, `app.js`, `styles.css`, `manifest.webmanifest`, `sw.js` и `_redirects`.
3. Откройте ссылку в Safari.
4. Нажмите «Поделиться» → «На экран Домой».

## Хранение данных

Данные и вложения сохраняются локально на устройстве. Автосохранение выполняется каждые 30 секунд и при сворачивании приложения. Облачной синхронизации между устройствами пока нет.

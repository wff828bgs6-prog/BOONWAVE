# BOONWAVE v6.0.54 — ENTRY, ACTIONS & MEDIA POLISH

Тестовая стабилизационная ветка на основе v6.0.52.

Исправления:
- одна цельная выезжающая панель с тремя действиями для всех карточек;
- карточка при выборе не сжимается и не меняет внутреннюю геометрию;
- корректный мягкий отклик строк «Я сейчас» без сдвига вправо;
- безопасный необязательный виброотклик там, где он поддерживается;
- выравнивание пустых плиток «Добавить материал»;
- компактная типографика кнопок основных материалов;
- явная обратная связь при выборе и добавлении файлов;
- повторная стабилизация первого кадра и связей.

Версия остаётся тестовой до проверки на iPhone Safari.

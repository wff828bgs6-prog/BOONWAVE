# BOONWAVE v6.0.45 — RELEASE FOUNDATION

Основа: v6.0.43-smooth-canvas-pan.

## Изменения

- безопасная обработка ошибок локального сохранения;
- версия схемы данных повышена до 7;
- изображения до сохранения уменьшаются до 2048 px и сжимаются;
- ограничение одного загружаемого файла: 30 МБ;
- вложения получают ownerId и экспортируются только для текущего профиля;
- импорт проверяет приложение, версию схемы, структуру и размер копии;
- удаление вложений ограничено текущим профилем;
- во время pan/zoom/drag декоративные слои кометы не создаются;
- комета сокращена до трёх слоёв;
- невидимые связи за пределами экрана не отрисовываются;
- исправлены CSS-селекторы режимов pan/zoom;
- «Процесс» убран из корневой нижней панели и остаётся связанной ветвью проекта;
- согласованы версии HTML, JS, CSS и service worker.

## Статус

Это проверочная release-ветка. Она не заменяет эталонную v6.0.43 до подтверждения владельцем продукта после тестирования.


## v6.0.45
Карточки по типам, квадратная карточка человека, встроенные действия, смысловой зум −50/0/+50, свободная камера, исправленный слой ветвей, режимы фото «Заполнить/Показать целиком».

# BOONWAVE Workspace v5.4.6

Новая тестовая версия BOONWAVE с акцентом на архитектуру карточек и удобство использования.

## Что реализовано

1. Сохранён утверждённый экран загрузки и форма входа/регистрации из v5.4.4.
2. Кнопка **«Визуальное знакомство»** продолжает работать как обход авторизации и сразу открывает рабочее пространство.
3. В верхней панели:
   - логотип-знак без надписи;
   - переключение **«Личное / Проекты»**;
   - меню настроек.
4. В рабочем пространстве:
   - кнопки **«Корневой элемент / Вписать / В центр»**;
   - нижняя навигация **«Древо / Сегодня / Результаты / Архив»**;
   - иконки выполнены в утверждённой стилистике;
   - плавные карточки с тач-эффектами.
5. Реализованы сущности:
   - Проект;
   - Человек;
   - Идея;
   - Цель;
   - Этап.
6. Реализована архитектура человека:
   - отдельная постоянная карточка человека;
   - отдельный проектный слой `projectPeople`;
   - задачи внутри проектной связи человека.
7. Для **проекта** добавлено фото карточки.
   - В свёрнутом состоянии проект отображается как визуальная иконка на основе фото.
8. Реализованы:
   - ручное перемещение карточек;
   - масштаб кнопками `+ / -`;
   - кнопка **«В центр»** фокусирует корневой граф;
   - кнопка **«Вписать»** подбирает масштаб для всех карточек.
9. Жест на карточке:
   - если двумя пальцами развести карточку — свёрнутая карточка разворачивается,
   - раскрытая открывается в режим подробной карточки.
10. Добавлены боковые панели **Сегодня / Результаты / Архив**.

## Что ещё можно развивать следующими шагами

- полноэкранное редактирование проектной карточки со всеми блоками бюджета и оплат;
- прикрепление PDF и файлов;
- поиск по людям, технологиям и тегам;
- нативные haptic feedback для iOS-оболочки;
- многопроектные рабочие столы с выбором стола;
- совместная работа нескольких пользователей.

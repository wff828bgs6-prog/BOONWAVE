# BOONWAVE — Version 5 Expanded

Расширенная мобильная PWA-ветка, воспроизводящая базовую логику BOONWAVE iPhone MVP v5:
пустой старт, отдельные пространства, локальное хранение, дерево, вложения, офлайн-режим.

## Реализовано

- пространства «Личное» и «Работа»;
- корневые элементы: проект, цель, человек, идея; этапы как связанные элементы;
- мягкие волновые связи;
- свободное перемещение карточек;
- удержание для контекстной панели;
- двойное нажатие для фокусировки;
- pinch-to-zoom и перемещение рабочего стола;
- три уровня масштаба карточек;
- проект: клиент, адрес, бюджет, аванс, остаток, срок, задачи, файлы, быстрые затраты и таблица затрат;
- человек: специализация, теги, контакты, сайт, соцсеть, зона внимания и проектные задачи;
- идея: источник, ссылка, личная мысль, теги и связь с проектами;
- цель: срок, прогресс и измеримый показатель;
- экраны «Сегодня», «Поиск» и «Входящие»;
- локальное автосохранение;
- экспортная структура данных совместима с дальнейшим расширением;
- PWA и офлайн-кэш;
- автоматическое обновление service worker.

## Запуск

Загрузите содержимое папки на Netlify так, чтобы `index.html` находился в корне.
На iPhone откройте адрес в Safari: «Поделиться» → «На экран Домой».

## Ограничения текущей сборки

- данные хранятся локально в браузере устройства;
- бинарное содержимое вложений пока не сохраняется, фиксируются имена и типы файлов;
- облачная синхронизация и совместная работа пользователей не подключены;
- импорт/экспорт JSON и полноценное IndexedDB-хранилище нужно перенести из оригинального кода Version 5 при наличии исходного архива.

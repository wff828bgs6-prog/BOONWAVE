# BOONWAVE v6.0.56 — ACTIONS & PERSON SYSTEM REFINE

Реализовано:
- отдельное состояние `actionsOpenId` для выезжающей вкладки;
- вкладка больше не зависит от `.selected`;
- убран `clip-path`, оставлены только transform + opacity;
- короткий тап открывает/закрывает вкладку, drag закрывает её, тап по полю закрывает;
- одна вкладка содержит три кнопки;
- в карточке «Человек» переработаны контактные строки;
- добавлены активные задачи и связанные сущности;
- обновлены статусы взаимодействия;
- добавлен переход из задачи человека в рабочий процесс с разовым мягким выделением;
- уменьшена жирность кнопок работы с фото.

# BOONWAVE v6.0.1-fixed

Корректирующая сборка.

Основа: BOONWAVE v6.0.0 без изменения загрузки, логотипов, компоновки, структуры экранов и общего CSS.

Единственное изменение:
- В режиме редактирования карточки «Проект» добавлена настройка позиционирования заглавного изображения: выбор изображения, масштаб, X/Y, Fit, Fill, сброс.

Параметры сохраняются в карточке проекта:
- coverAssetId
- coverScale
- coverOffsetX
- coverOffsetY

Оригинальные изображения не изменяются.

# Команды service-слоя рабочего процесса

Реализованы атомарные операции одного process Node:

- выбор активного этапа;
- добавление и перемещение этапа;
- архивирование и восстановление этапа;
- добавление задачи в выбранный этап;
- изменение порядка задачи внутри этапа;
- перенос задачи в другой этап;
- архивирование и восстановление задачи.

Каждая команда сначала формирует и нормализует новую карточку, затем сохраняет её через storage adapter и только после успешной записи обновляет store.

# BOONWAVE — варианты панели управления

Дата фиксации: 2026-07-04
Ветка: `codex/control-panel-standard`
Раздел: Панель управления

## Вариант 1 — текущая утверждаемая база

Статус: зафиксирован как контрольная точка для возврата.

Этот вариант соответствует состоянию панели после ручной проверки на iPhone Safari:

- панель может быть справа, слева и снизу;
- внизу панель компактная, с видимым ползунком масштаба;
- порядок кнопок: Главный обзор, Добавить, Контакты, Замок, Дополнительные действия;
- иконка главного обзора — четыре стрелки внутрь;
- кнопки одной яркости и одной визуальной серии;
- окно «Дополнительные действия» компактное, с блоками: Связь, Архивировать, Архив, Отменить, Повторить;
- блок «Положение панели» содержит: Слева, Справа, Снизу;
- архивный слой подключён: карточку можно архивировать, открыть архив и восстановить;
- старая системная карточка «Я Есмь» скрыта с рабочего стола, а системное название заменяется на «Моя Вселенная».

Визуальные параметры варианта 1:

- нижняя панель: около 92 px высоты;
- зона ползунка: около 34 px;
- ряд кнопок: около 41 px;
- окно «Дополнительные действия»: крупный заголовок, читаемые кнопки, отключённые действия приглушены;
- стиль: тёмная полупрозрачная основа, тонкая обводка, мягкое свечение, премиальная темная система.

Если изменения варианта 2 не понравятся, нужно вернуть визуальные параметры окна «Дополнительные действия» к варианту 1.

## Вариант 2 — экспериментальная полировка

Статус: следующий эксперимент.

Цель:

1. Сделать заголовок «Дополнительные действия» чуть спокойнее, чтобы он не конкурировал с карточками.
2. Сделать отключённые кнопки более «дорогими»: не грязно-тусклыми, а чисто неактивными.

Важно: вариант 2 не должен менять функциональность панели. Это только визуальная полировка CSS.

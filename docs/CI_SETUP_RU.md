# BOONWAVE CI

GitHub Actions проверяет каждый pull request в `main` и каждый push в `main`.

Матрица проверки:

- Node.js 20;
- Node.js 22.

Для каждой версии выполняются:

1. установка зависимостей;
2. проверка целостности репозитория;
3. проверка синтаксиса;
4. полный набор тестов.

Команда проверки:

`npm run check`

До появления зелёного результата CI pull request не считается готовым к merge.

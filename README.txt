BOONWAVE TEST MVP — запуск

1. Для Netlify: загрузите содержимое этой папки или ZIP целиком.
2. Для локального теста нужен HTTP-сервер, например:
   python3 -m http.server 8080
   Затем откройте http://localhost:8080
3. На iPhone после публикации: Safari → Поделиться → На экран «Домой».

Что реализовано:
- анимация BOONWAVE;
- локальная регистрация и вход;
- приветствие после первого входа;
- создание личного и рабочего пространства;
- автоматическое ядро «Я» для личного пространства;
- создание ядра «Проект» или «Цель» для рабочего пространства;
- связанные элементы и мягкие волновые связи;
- свободное перетаскивание карточек;
- переключение между пространствами;
- локальное сохранение;
- экспорт и импорт JSON;
- PWA, офлайн-кэш и автоматическое обновление service worker.

Важно: это тестовая локальная версия. Пароли хранятся в браузере без серверного шифрования и не предназначены для публичного production-запуска.

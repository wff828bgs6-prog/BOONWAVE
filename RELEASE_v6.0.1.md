# BOONWAVE v6.0.1

Изменение: в режиме редактирования карточки «Проект» добавлен блок настройки заглавного изображения.

## Что добавлено

- выбор заглавного изображения из изображений проекта;
- предпросмотр обложки;
- настройка масштаба;
- настройка позиции X и Y;
- кнопки «По размеру», «Заполнить», «Сбросить позицию»;
- сохранение параметров `coverScale`, `coverOffsetX`, `coverOffsetY`;
- отображение сохраненной позиции обложки на карточке и в детальном просмотре.

## Проверка

1. Открыть «Визуальное знакомство».
2. Открыть проект.
3. Перейти в «Редактировать».
4. Добавить изображение в «Основные материалы».
5. Настроить «Заглавное изображение».
6. Сохранить карточку.
